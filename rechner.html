RECHNER

BESCHREIBUNG

Hier können Sie verschiedenen operationen machen. Dieser Bildschirm gibt zwei display. Ein Größ für den aktuell wert. Ein klein für alle Werte und operationen speichern.
Wenn Sie falsch sind, können Sie die «CE» Taste nutzen für ganz wert im Größen Display löschen. Sie können die Gelb Taste für nur eine Ziffer löschen.
Wann Sie vertig sind, können Sie die Grüne Taste klicken und dann kommt das Ergebnis.
